# Learn Java - Programming Journey

A comprehensive learning repository for mastering Java fundamentals and object-oriented programming concepts. This repository contains course projects and exercises from **Tim Buchalka's Learn Programming Academy** course.

## 📚 Overview

This project serves as a progressive learning path through Java programming, starting from basic concepts and advancing to more complex object-oriented design patterns. Each section builds upon previous knowledge with practical exercises and real-world application projects.

## 📂 Repository Structure

### **Section 5 - Methods & Fundamentals**

- Method creation and overloading
- Passing parameters and return values
- Method challenges and exercises

### **Section 6 - Control Flow**

- Loop implementations (for, while, do-while)

- Challenges with loops and conditional logic

### **Section 7 - Introduction to OOP**

- Class and object basics
- Constructors and initialization
- **Key Topics:**
    - `BankAccount.java` - Class encapsulation fundamentals

    - The `super` keyword
    - String manipulation
    - The Object class
    - Record types
    - Working with complex numbers

### **Section 8 - Advanced OOP Concepts**

- **Encapsulation** - Data hiding and access control
- **Composition** - Building complex objects
- **Polymorphism** - Method overriding and interfaces

### **Section 9 - Arrays & Collections**

- Single-dimensional arrays

### **Section 10 - Collections Framework**

## 🚀 Getting Started

1. **Prerequisites:**
    - Java Development Kit (JDK 11 or higher)
    - IntelliJ IDEA or your favorite Java IDE
    - Basic understanding of programming concepts

2. **Running Projects:**
    - Open the desired project folder in your IDE
    - Locate the `Main.java` file
    - Run the main method

3. **Studying the Code:**
    - Review each section sequentially
    - Run the projects to see output
    - Modify and experiment with the code
    - Complete all challenges before moving forward
